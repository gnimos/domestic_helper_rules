# 一、道德及個人操守
1. 根據私隱條例，嚴禁在僱主家中任何地方拍照或攝錄（包括拍照或攝錄家中環境、成員、寵物），以及對任何人透露僱主家中環境及狀況，或將家中情形分享至網絡上。  
2. 嚴禁在僱主的睡床、沙發、椅子等地方坐卧，以及私自濫用僱主衣服和個人物品。  
3. 僱主會提供工作電話。工作時嚴禁使用私人手提電話作私人用途；休息或用膳時方可使用。  
4. 工作時嚴禁戴上耳機聽歌。  

---

# 二、家務
1. 每天必須按照工作表上的指示工作。  
2. 每天早上 5 時起床，並需完成所有家務後方可睡覺。  
3. 除非得到僱主同意，否則嚴禁關門工作。  
4. 洗衣服前必須將反面衣服翻回正面。  
5. 衣服晾乾後，衣架必須放回指定儲物櫃。  
6. 睡前更換地拖筒水，清潔地拖及掃帚。  
7. 清潔時物件需搬起、抹乾淨，再放回原位。  
8. 各地方需使用不同抹布；每週需消毒及清洗所有抹布。  
9. 如發現塵埃或骯髒地方，即使已完成家務仍需立即清潔。  
10. 僱主回家時，必須立即將鞋子收回鞋櫃，以免狗咬破。  

---

# 三、清潔及個人衛生
1. 嚴禁把私人物品放在客廁（除浴巾外）。  
2. 一天只可洗澡一次。  
3. 僱主只提供一次沐浴露及牙膏，用後需自行購買。  
4. 僱主不提供其他個人用品（牙刷、洗衣液、潤膚膏等）。  
5. 私人衣物須手洗，不可與僱主衣物混合。  
6. 私人物品必須放於五桶櫃，不可放在其他地方。  

---

# 四、食物及膳食安排
1. 僱主提供膳食；僱主給什麼就吃什麼。  
2. 僱主吃飯時間不定，傭人可自行煮食。  
3. 吃飯前必須先清洗煮食爐具及廚具。  
4. 只可在廚房或雜物房進食。  
5. 未經允許不得擅取任何食物；如糧食將完需提前一天通知僱主。  
6. 嚴禁將私人食物放入僱主雪櫃。  
7. 必須使用電磁爐煮食，不可使用明火。  
8. 僅可使用僱主指定之餐具。  

---

# 五、寵物照顧
1. 狗隻咬物時要立即安全制止並保留損毀物件回報。  
2. 狗隻吠叫需立即制止。  
3. 每天替狗隻梳毛。  
4. 狗隻如廁後需立即清理，並抹乾淨地板；大便後需清潔狗隻屁股。  
5. 每天清潔兔籠。  
6. 每天清潔鳥籠外面及膠片。  
7. 狗隻外出回家後需用濕紙巾清潔腳部。  

---

# 六、嬰兒照顧
- 訓前要用嬰兒洗潔精清洗所有用過食品用具然後蒸氣消毒，例如碗，匙，奶樽
- 清洗完食品用具要放回BB車上
- 一蓋奶粉兌換30ml熱水
- 要按照時間表時間照顧BB，不可自行加減餵食次數，不可自行更改項目時間
- 每天睡覺前要清潔及收拾嬰兒房，包括用品及玩具放回原位




1. 接觸嬰兒前需以洗手液洗手。  
2. 禁止讓嬰兒看電話或電視。  
3. 必須嚴格遵從時間表，並記錄於 WhatsApp 群組。  
4. 餵食前需替嬰兒抹手。  
5. 餵食前需先試溫度。  
6. 餵食時不可離開嬰兒。  
7. 餵食時嬰兒必須坐於嬰兒椅並扣上安全帶。  
8. 食後需用暖水紗巾輕印清潔嬰兒的口與手。  
9. 餵食時間需控制在 15–20 分鐘。  
10. 嬰兒拒食時不可強迫。  
11. 每次餵食完需記錄時間及拍照回報。  
12. 嬰兒膳食由僱主準備，不可擅自添加。  
13. 嬰兒房需準備可供嬰兒自行拿取之水樽。  
14. 不可對嬰兒說「不」或責罵。  
15. 不可將嬰兒帶離房間。  
16. 嬰兒僅可玩遊戲架上的玩具。  
17. 清潔嬰兒用品需用指定抹布。  
18. 照顧嬰兒期間嚴禁使用手機。  
19. 危險物品需遠離嬰兒。  
20. 嬰兒發生意外時必須立即通知僱主。

# 七、家規
1. 每天 5 時起床，未完成工作需即日匯報。  
3. 願意假日工作者會獲得額外薪金。  
4. 平日無需外出購買日用品。  
5. 放假可於 08:20 出門，並需於最遲 18:30 回家；否則需自費搭的士。  
6. 損毀僱主財物需回報，僱主可扣薪處理。  
7. 嚴禁隨便進入主人房（除清潔外）。  
8. 取車匙、門匙前需獲允許。  
9. 所有物件需用後歸位。  
10. 有客人時需奉茶並保持友善。  
11. 未經指示不可開門，即使是送貨員。  
12. 嚴禁擅自開空調。  
13. 工作需勤快、態度良好。  
14. 要誠實，不可說謊。  
15. 節省能源，不可亂開燈／冷氣。  
16. 嚴禁借錢或做擔保人。  
17. 嚴禁盜竊。  
18. 嚴禁帶朋友到僱主家。  
19. 嚴禁外宿。  
20. 嚴禁擅自離港。  
21. 嚴禁從事違法活動。
